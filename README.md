## How to Run the Project

1. Clone this repository:

```bash
git clone https://github.com/your-username/your-repo.git
```
2. Navigate to the project directory

```bash
cd PokeChallenge
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
``
